# Coin Runner

Canvas-based infinite runner built with Next.js App Router.

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000 to play. Space or tap to jump, Enter to start/restart.

## Deploy to Vercel

1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the repo into Vercel.
3. Keep the default Next.js build settings (`npm run build`).
4. Deploy.

High scores are stored in `localStorage` on the client.
